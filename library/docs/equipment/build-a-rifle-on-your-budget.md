# Build a Rifle on *Your* Budget

Rifle Setups - 2 use cases, 3 tiers

This guide's purpose is to be concise and present different tiers of rifle setups for the home defense and patrol use case. The "why" will be covered somewhat less but perhaps the document will be expanded later to include more on rationale

Note that reference links are just reference - for many of these products you will be able to find a slightly different SKU for a better price, so make sure to check which of the similar SKUs are actually the best deal.

### Tier #3: Premium

Price Range: ~$2400-4000
This tier is past the diminishing returns point but there are things about these options that are genuinely nicer. If you're willing to spend as much as makes sense to spend for something very nice, go with this tier; otherwise, save your money. More expensive options than this tier exist but are an extremely questionable value. These setups can be pieced together to save money but for simplicity this will be mostly complete rifles.

### Tier #3: Premium

Price Range: ~$3100-6000
This tier is past the diminishing returns point but there are things about these options that are genuinely nicer. If you're willing to spend as much as makes sense to spend for something very nice, go with this tier; otherwise, save your money. More expensive options than this tier exist but are an extremely questionable value. These setups can be pieced together to save money but for simplicity this will be mostly complete rifles.
